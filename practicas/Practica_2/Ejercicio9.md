9. ¿Qué diferencia hay entre el uso de `include` y `extend` a la hora de incorporar un módulo en una clase?

  a. Si quisieras usar un módulo para agregar métodos de instancia a una clase, ¿qué forma usarías a la hora de
     incluirlo en la clase?

  b. Si en cambio quisieras usar un módulo para agregar métodos de clase, ¿qué forma usarías a la hora de incluir el
     módulo en la clase?

a. `include`

Al incluir el modulo en una clase, los metodos se agregan como `metodos de instancia`

b. `extend`

Al incluir el modulo en una clase, los metodos se agregan como `metodos de clase`
